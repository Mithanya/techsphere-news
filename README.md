# TechSphere News Component

## Overview

TechSphere is a responsive news UI component built using HTML and CSS. It demonstrates floating layouts, scrollable content sections, and modern design principles. The component is lightweight, mobile-friendly, and designed for real-world layout handling.

## Features

- Floating image layout with structured content
- Scrollable live updates section using overflow control
- Responsive design for all screen sizes
- Clean and modern UI with smooth styling
- Animated live indicator
- Organized content blocks for readability

## Technologies Used

- HTML5
- CSS3 (Flexbox, Float, Media Queries, Animations)
